ტესტ ქეისების პასუხისმგებლობის განაწილება

პროექტზე ვიმუშავე მარტო და ყველა ტესტის ავტომატიზაცია შესრულებულია ჩემს მიერ. განხორციელებული UI ტესტებია: Login, Logout, Register, Add to Cart, Remove from Cart, Search Product, View Product Details, Contact Form, Subscription და Checkout Flow. ასევე ჩემ მიერ არის დაწერილი API ტესტები: Get User, Create User, Update User, Delete User, List Users, Get Products, Create Product, Update Product, Delete Product და Authentication.

ტესტების გაშვების და რეპორტების გენერირების ინსტრუქცია

პირველ რიგში საჭიროა დაინსტალირებული იყოს Java და Maven. შემდეგ უნდა გაიხსნას ტერმინალი პროექტის მთავარ ფოლდერში, სადაც მდებარეობს pom.xml ფაილი. პროექტის გასაწმენდად გამოიყენება ბრძანება mvn clean, ხოლო ტესტების გასაშვებად mvn test. შესრულების შემდეგ შედეგები გენერირდება target ფოლდერში. თუ გამოყენებულია Allure Report, მისი სანახავად გამოიყენება ბრძანება allure serve target/allure-results.
